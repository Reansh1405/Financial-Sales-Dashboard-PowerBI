## 📌 Project Overview

The **Financial Sales Dashboard** is an interactive Business Intelligence solution developed using **Microsoft Power BI** to transform raw financial data into meaningful business insights.

The dashboard enables users to analyze sales performance, profitability, product performance, and geographical distribution through interactive visualizations and KPI cards. It demonstrates practical applications of Power BI, Power Query, and DAX in solving real-world business reporting scenarios.

---

## 🎯 Project Objectives

- Analyze overall business sales performance.
- Monitor total revenue and profitability.
- Identify the top-performing products based on profit.
- Compare sales across different countries.
- Visualize sales by customer segments.
- Build an interactive dashboard using Power BI.
- Apply Power Query for data transformation.
- Create business KPIs using DAX.
